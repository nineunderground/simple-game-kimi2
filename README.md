A simple, elegant sliding tile puzzle game built as a single-page application.

## Play

Open `index.html` in your browser or visit the GitHub Pages site.

## Features

- **Classic 15-puzzle gameplay** - Arrange tiles 1-15 in order
- **Move counter & timer** - Track your performance
- **Keyboard support** - Use arrow keys to play
- **Responsive design** - Works on desktop and mobile
- **Smooth animations** - Satisfying tile movements
- **Always solvable** - Shuffling uses valid moves

## How to Play

1. Click **Shuffle** to scramble the tiles
2. Click tiles adjacent to the empty space to move them
3. Arrange numbers 1-15 in order (empty space at bottom-right)
4. Try to solve it in the fewest moves!

## Controls

- **Mouse/Touch**: Click on tiles to move them
- **Keyboard**: Use arrow keys to slide tiles

## Run with Docker

```bash
docker compose up --build
```

Then open http://localhost:8080

## Tech

Pure HTML, CSS, and JavaScript. No dependencies. Just open and play!
